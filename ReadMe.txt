Cette intégration à pour but de récupérer l’ensemble des paramètres d’une installation solaire thermique SoliAsrt et des les gérer comme des Sensors Home Assitant.
Normalement la documentation est assez précise. 
L’installation demande un minimum de connaissances Home Assistant : Copier, Modifier un fichier ou des éléments Yaml. Normalement rien d’extraordinaire.
Cette intégration est conçue pour Home Assistant, néanmoins il sera assez facile de l’adapter à d’autre système de domotique (voir le chapitre principe de fonctionnement)
Pour en faciliter l’installation et la maintenance, cette intégration est fournie sous forme d’un package HomeAssistant.
Commencez par lire le fichier pdf de documentation.
